# DragonBoard Vision

DragonBoard adalah Virtual Tabletop yang dibuat khusus untuk Dragonbane.

Tujuan utamanya adalah membantu permainan tabletop secara langsung (face-to-face), bukan menggantikan pengalaman bermain fisik.

Prinsip utama:

- Tablet First
- Dragonbane First
- Offline Friendly
- Free Forever
- Fast and Simple

DragonBoard tidak bertujuan menjadi pengganti Foundry VTT.

DragonBoard bertujuan menjadi companion tool untuk Game Master Dragonbane.
